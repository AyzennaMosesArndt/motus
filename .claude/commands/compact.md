# /compact
Compress current context before switching to a new major task.

## When to use
- Before switching from pipeline work to frontend work
- Before starting a new feature after a long debugging session
- When context feels polluted with old file contents

## Steps
1. Summarize what was accomplished in this session (3–5 bullet points)
2. Note any open issues or TODOs
3. Write summary to CLAUDE.local.md under "## Last Session"
4. Trigger Claude Code native /compact to compress context window
